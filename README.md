# 🏭 Mini Factorio

Мини-игра в стиле Factorio — стройте автоматизированную фабрику, добывайте ресурсы, создавайте конвейерные линии и производите пластины!

## 🎮 Описание

Размещайте здания на карте с залежами руды и угля. Буры добывают ресурсы, конвейеры транспортируют их, печи переплавляют руду в металл, а сборщики создают пластины из металла и угля. Автоматизируйте всё!

## ✨ Возможности

### Здания

| Здание | Описание |
|--------|----------|
| ⛏️ **Бур** | Добывает руду или уголь из залежей. Вращайте для настройки направления выхода |
| 🔄 **Конвейер** | Транспортирует предметы между зданиями. Соединяйте конвейеры в цепочки |
| 🔥 **Печь** | Переплавляет руду в металл. Принимает руду и уголь с конвейера |
| 🏭 **Сборщик** | Производит пластины из металла и угля. Принимает ресурсы с конвейера |
| 📦 **Сундук** | Хранит до 100 предметов (до 50 каждого типа). Накапливает ресурсы |

### Ресурная цепочка

```
⛏️ Бур (руда) → 🔄 Конвейер → 🔥 Печь → 🔄 Конвейер → 🏭 Сборщик → 🔄 Конвейер → 📦 Сундук
⛏️ Бур (уголь) → 🔄 Конвейер ↗                          ↑
                                          └─ металл + уголь ─┘
```

### Визуальные эффекты

- Анимированные шестерёнки в сборщике
- Огонь и дым из трубы печи
- Вращающаяся буровая головка с вибрацией
- Плавное движение предметов по конвейеру
- Вспышка «+1» при производстве пластины
- Жёлтая анимация при вращении зданий
- Подсветка зданий при наведении курсора

### Информационная панель

При наведении на любое здание отображается тултип с:
- Входным и выходным буферами (с детализацией по типам ресурсов)
- Прогрессом текущей операции
- Статусом работы («Производит пластины», «Нет металла» и т.д.)
- Содержимым хранилища (для сундука)
- Предметами на тайле конвейера

## 🕹️ Управление

| Клавиша | Действие |
|---------|----------|
| **ЛКМ** | Построить выбранное здание |
| **R** | Вращение здания под курсором / направление нового конвейера |
| **Пробел** | Пауза / продолжить |

## 🚀 Запуск

Просто откройте файл `index.html` в любом современном браузере:

```bash
# Откройте файл в браузере
open index.html        # macOS
xdg-open index.html    # Linux
start index.html       # Windows
```

Сервер не требуется — игра работает полностью в браузере.

## 🎯 Как играть

1. **Найдите залежи** — коричневые круги = руда, тёмные = уголь
2. **Поставьте буры** — кликните на залежи, выберите «Бур»
3. **Проложите конвейер** — от бура к печи и далее
4. **Настройте направление** — наведите на здание и нажмите `R`
5. **Запустите производство** — печь переплавит руду, сборщик сделает пластины
6. **Собирайте ресурсы** — сундук накапливает и переносит в инвентарь

## 💡 Советы

- Ставьте бур так, чтобы его выход смотрел на конвейер
- Соединяйте несколько конвейеров в цепочку для транспортировки на расстояние
- Печь принимает руду и уголь, сборщик — металл и уголь
- Сундук вместит до 100 предметов — идеально для хранения излишков
- Тултипы покажут содержимое любого здания и его статус
